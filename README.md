# AMS595-C-Assignment7
Project 1: Basic C++ Syntax
AMS 595 / DCS 525

Author: Yusha Bai


Compilation Instructions
====================================

This project was written in standard C++.

To compile the program, run:

g++ main.cpp -o project1


Execution Instructions
====================================

After compilation, run the executable using:

./project1


Description
====================================

This program demonstrates basic C++ syntax and programming concepts,
including:

1. Conditional statements (switch-case)
2. While loops
3. User-defined functions
4. Vector operations
5. Prime checking
6. Factorization and prime factorization
7. Iterative construction of Pascal's Triangle

All functions are implemented independently and are fully commented
for clarity.
